# IOT(사물인터넷)
- The infrastructure of the information society
- Iot는 사람, 사물, 공간, 데이터 등 모든 것이 인터넷으로 서로 연결되어 정보가 생성, 수집, 공유, 활용 되는 초 연결 인터넷
- 인터넷에 연결된 컴퓨팅 기능을 가진 장치들이 센서를 통해 정보를 감지, 목적지에 전달하거나 원격으로 장치들을 통제될 수 있는 환경 -> 편리성, 효율성, 정확성 증대
- 컴퓨팅 능력을 갖춘 기기들(things)이 인터넷과 같은 거대한 망에 연결, 하나의 틀로 묶어 제공하는 서비스에 대한 기술

# IOT 적용 기술
- 임베디드 시스템
  - 컴퓨팅 능력을 갖춘 기기가 내장된 시스템
  - 스마트폰, 네비게이터, 자동차, 항공기, 의료기기 등 범용 컴퓨터를 제외한 컴퓨팅 시스템이 내장된 모든 시스템
- 프로그램
- 인터넷
- 센서
- 보안
- 사물들이 인터넷에 연결되어 있음으로 인해 해킹의 대상이 됨

- 데이터 수집 -> 데이터 전송 -> 통신 -> 데이터 분석 -> 활용
- 센서 (실제 데이터를 수집하는 최 접점)
- 임베디드 단말(수집된 정보를 취합 및 전송)
- 네트워크(근거리 : 주로 센서와 임베디드 단말 통신, 원거리 : 단말과 서버 사이의 통신)
- 보안 (데이터 유출, 외부 침입, 사생활 침해 등 방지)
- 미들웨어 (Event processing : 대량 데이터에 대한 1차 패턴 분석, Messaging : 데이터의 전송 보장)
- 클라우드 컴퓨팅 (효율적인 End-to-End IOT 플랫폼 구축)
- 데이터 처리 및 분석(빅데이터 : 센서에서 수집된 비정형, 대량 데이터 처리, DW/BI :데이터로부터 의미 있는 비즈니스 정보 추출)

- 소형화, 저전력화, 표준화, 저가격화
 
# 보안
- 단말 : 분실/도난, 물리적 파괴
- 네트워크 : 무선신호 교관, 정보 유출, 데이터 위변조, 서비스 거부
- 애플리케이션 : 정보유출, 데이터 위변조, 서비스 거부

# 사생활 보호
- 사물을 식별하기 위한 식별자 관리 -> 직접적인 사생활 침해 위험
- 성향 데이터 분석을 통한 개인 사용자 추적
- 법규와 도덕적인 가치관을 통해 극복

# 관리체계
- 무수히 많은 사물들과 그것을 사용하는 인간들의 상호작용이므로 전체적으로 제어할 수 있는 체계가 필요
- 통제되지 않는 기술(보안, 프라이버시에 대한 고려) 발전은 재앙을 초래
- 개인(긍정적인 참여), 기업(기술 발전과 사회에 대한 책임), 정부(법규 및 정책) 상호 노력
